# Charte Graphique - Pixel Adventure

Basée sur l'analyse visuelle de l'écran de démarrage "Chiadé".

## 1. Palette de Couleurs (Neon-Tech)

| Élément | Couleur | Hex | RGB | Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Primaire** | Cyan Électrique | `#00FFFF` | `(0, 255, 255)` | Joueur, lueur, interactions clés |
| **Secondaire** | Violet Profond | `#8A2BE2` | `(138, 43, 226)` | Murs, structures technologiques |
| **Accent 1** | Rose Magenta | `#FF1493` | `(255, 20, 147)` | Sortie (Exit), points d'intérêt |
| **Accent 2** | Orange Slime | `#FF8C00` | `(255, 140, 0)` | Ennemis basiques, dangers |
| **Accent 3** | Jaune Éclair | `#FFFF00` | `(255, 255, 0)` | Projectiles, yeux d'ennemis |
| **Fond** | Bleu Abysse | `#0A0A1E` | `(10, 10, 30)` | Arrière-plan, menus |
| **Ombre** | Noir Cobalt | `#050510` | `(5, 5, 16)` | Bordures de murs, vides |

## 2. Style Visuel

*   **Atmosphère** : "Cyber-Dungeon". Un mélange de labyrinthe classique et de technologie futuriste (néons, circuits).
*   **Rendu "Chiadé"** :
    *   Utilisation intensive de **lueurs (glow)** et de gradients.
    *   Tuiles de murs avec motifs de "circuits" ou de briques technologiques.
    *   Sol avec de légères particules de lumière.
*   **Sprites** : 
    *   Personnage avec une armure émettant de la lumière.
    *   Ennemis avec des yeux lumineux contrastant avec leur corps sombre.

## 3. Typographie

*   **Titre** : Style "Retro-Gaming" / Pixel Font épaisse avec contour néon.
*   **Interface** : Police sans-serif propre (type Roboto/Inter) ou pixel art lisible.
*   **Feedback** : Texte cyan pour les actions positives, magenta pour les alertes.

## 4. Effets Spéciaux

*   **Scanlines** : Légères lignes horizontales (optionnel) pour l'effet moniteur rétro.
*   **Pulsations** : Les éléments cyan doivent pulser légèrement en intensité.
*   **Particules** : Émissions de pixels lors des collisions ou de la mort.
